Custom select

Syntax

To create custom select u need to call new CustomSelect(container,config); U can work with custom select just like with native
container - place where select will be inserted;
config - object, with properties:

  • options - array of object with properties title and value;
  • defaultOption - default title of selector

Events

  • show - occur exactly after options showed;
  • hide - occur exactly after options hided;
  • change - occur exactly after select value changed (after click or enter).

Methods

  • customSelect.show() - show options;
  • customSelect.hide() - hide options;
  • customSelect.toggle() - show/hide options.
