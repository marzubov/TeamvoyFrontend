Calendar Demo

Syntax

To create calendar you need to call:
new Calendar(container,config);
container - required parameter, the place where calendar will be inserted.
config - optional parameter, object with fields:
  • year - current year, type: number;
  • month - current month, type: number;
  • firstDayOfWeek - first day of week in calendar, only english language, type: string;
  • locale - language of calendar, short name, type: string. Example: 'monday'.

Methods

  • myCalendar.showToday(); - set calendar config to show current day;
  • myCalendar.on(eventName,callback); - add new function to event listener;
  • myCalendar.off(eventName,callback); - remove function from event listener;
  • myCalendar.trigger(eventName); - call event.

Event List

  • onMonthChanged - occur when config month data changed
