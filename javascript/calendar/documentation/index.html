Calendar Demo

Syntax

To create calendar you need to call:
new Calendar(container,config);
container - required parameter, the place where calendar will be inserted.
config - optional parameter, object with fields:
  • year - current year, type: number;
  • month - current month, type: number;
  • firstDayOfWeek - first day of week in calendar, only english language, type: string;
  • locale - language of calendar, short name, type: string. Example: 'monday'.

Methods

  • myCalendar.showToday(); - set calendar config to show current day;
  • myCalendar.on(eventName,callback); - add new function to event listener;
  • myCalendar.off(eventName,callback); - remove function from event listener;

Event List

  • onMonthChanged - occur when config month data changed

Examples:

Default calendar:
config = {}
You can style calendar with classes for each component: caption, days, weekends etc.
Classes are adding to the components in customize_calendar.js file
config = {year: 2014, month: 5, style:"customize", firstDayOfWeek: "Пн" , locale: "ua", weekends:["Сб","Нд"]}
You can change config property anytime with similar code:
myCalendar.config = newConfig;
It will reset data in calendar. Try It!
You can add more than 7 days in one row:
config.daysInWeek = 14;
myCalendar.config = {daysInWeek: 14};
It will set 14 days in one row.
If days in row is not multiply of 7, days names wont be shown
config.daysInWeek = 13;
myCalendar.config = {daysInWeek: 13};
It will set 13 days in one row and there wont be days names.
